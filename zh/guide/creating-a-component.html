# 创建一个组件

提示

此功能只能在安装完 npm 依赖包后方可使用。

# nj ac

进入已有项目目录后，输入如下命令创建新组件：

nj ac

# 请选择生成时要使用的组件模板
Which component template do you want to use?
> class component       # class组件
  function component    # 函数组件

# 输入新创建的组件名称
Component Name: newComponent

√ Created component completed!
