# 创建一个项目

## nj init

在当前目录下创建新项目目录：

```sh
nj init

# 请按上下键选择项目模板
Please select a project template:
> react-mobx      # 默认使用mobx
  react-mst       # 默认使用mobx-state-tree

# 输入新建的项目目录名称
Project Name: new-project

√ Download finished.

√ Unzip completed.

# 是否需要单元测试
Do you need jest? (Y/N)

# 是否需要 e2e 测试
Do you need e2e test? (Y/N)

# 是否需要 ts
Do you want to use typescript? (Y/N)

# 是否马上安装依赖包
Do you want to install packages? (Y/N)

√ Init completed!
```
