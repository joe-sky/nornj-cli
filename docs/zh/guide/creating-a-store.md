# 创建一个 Store

::: tip
此功能只能在安装完 npm 依赖包后方可使用。
:::

## nj as

进入已有项目目录后，输入如下命令创建新Store：

```sh
nj as

# 请选择生成时要使用的Store模板
Which store template do you want to use?
> mobx               # 使用mobx
  mobx-state-tree    # 使用mst

# 输入新创建的Store名称
Store Name: newStore

√ Created store completed!
```
