# 创建一个页面

::: tip
此功能只能在安装完 npm 依赖包后方可使用。
:::

## nj ap

进入已有项目目录后，输入如下命令创建新页面：

```sh
nj ap

# 请选择生成时要使用的页面模板
Which page template do you want to use?
> default  # CURD
  chart    # 图表
  form     # 表单
  empty    # 空页面

# 输入新创建的页面名称
Page Name: newPage

√ Created page completed!
```
